## typescipt
ts 来自微软，是js的超集，添加了类型约束
js 弱类型经常会出类型错误
- 静态的类型编译 ts->js文件可以实时检查类型或代码错误
- ts 非常强大，已经是AI Agent的标配
## js的易错性
- 浏览器input输入 我们以为是数字，但实际是字符串
- + 加法和字符串拼接
- 又不报错，
