## 正则表达式 Regular Expression
/ / 表示一个正则表达式
每次就匹配一个字符
{}表示匹配的次数
[]表示匹配的字符范围
\d表示匹配随机数字
^表示匹配字符串的开头
$表示匹配字符串的结尾

let str = '13888888888'
let reg = /^1[3-9]\d{9}$/

为什么13888888888ss,aa13888888888也是true？
❌ 不加 ^$ —— 只要字符串里"包含"就匹配
/1[3-9]\d{9}/.test(str)    

 ✅ 加上 ^$ —— 必须整个字符串完全匹配
/^1[3-9]\d{9}$/.test(str)
