 8种数据类型
 根据ecma262规定
 primitive type 原始数据类型有六种
 其中numeric包含number和bigint
 还有object对象 nonprimitive type

## 简单数据类型
 原始数据类型(primitive type)
 - 数值类型(number)
 - 字符串类型(string)
 - 布尔类型(boolean)
 - 空类型(null)
   表示某处应该有个值，不过目前没有（空的值）
   可以用于清空一个变量的值，释放内存，垃圾回收
   用于表示一个有意设置为空的对象引用(空的引用)
   举链表例子最直观：
   js
   node.next = null
   next 本该存「下一个节点」的引用；
   现在手动写成 null，明确表示：后面没有节点
 - 未定义类型(undefined)
    表示一个未初始化或者不存在的变量值
     -当申明一个变量但是未赋值时，默认值为 undefined
     -某个对象的属性不存在时，默认值为 undefined
     -函数没有返回值，默认值为 undefined
     -访问不存在的数组索引，默认值为 undefined
es6之前，js只有六种数据类型

## 内存分配去思考
- 冯诺依曼 现在计算机设备
 输入，输出，运算器，存储器，控制器
-代码 外存 硬盘
-编译，代码从硬盘调入内存之中
-执行上下文，变量环境，词法环境，执行代码 
-推入**调用栈** 在栈内存中
 栈内存 快，空间小，很珍贵
 变量环境/词法环境  原始数据类型，写入值
 引用数据类型，也在栈内存中，存的是对象的地址，指向堆内存中的对象
 编译阶段给他的空间正好合适
 函数执行上下文对象占用多少空间是计算的出来的，当一个函数执行完出栈，很快算出新的栈顶元素位置，指针的偏移量就可以切换上下文的执行，快速，稳定，扩展

 堆内存
 空间大、不固定
 分配晚、编译期算不出大小
 读写比栈慢
 不会自动释放
