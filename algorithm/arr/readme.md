## 数组的创建

-ADT 的认识

-push，unshift，shift，pop
 -修改原数组，破坏了原来的数组，这些方法都不是纯函数
 -push 返回新的length
 -pop 移除最后一个元素的值
 unshift 对头添加元素的值
 shift 对头移除元素的值

-纯函数

empty 数组的这个位置还没有被占据，不属于任何类型

##  遍历数组的方法、
-for
  机器化，命令式，缺点是可读性不好
  优点是性能好
-for of
  语义很好
-for in
-for each
  入调用栈 执行上下文
  优点是功能强大
  缺点是不能中断遍历
-map，filter，every，some
基于foreach
 返回一个全新的数组 纯函数
 1. map
 逐项加工，返回新数组，原数组不变。
 2. filter
 按条件筛选，返回符合条件的新数组。
 3. every
 判断全部元素是否满足条件，返回布尔值，一假就终止。
 4. some
 判断至少一个元素满足条件，返回布尔值，一真就终止。
 5. reduce
 从左到右，累计处理数组中的每个元素，返回一个值。
