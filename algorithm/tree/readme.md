## 树与二叉树
数据结构的树，是对现实世界中树的一层简化
1. 根节点：树最顶层节点，无父节点。
2. 父 / 子节点：直接相连的上下节点，上层为父，下层为子。
3. 叶子节点：没有子节点的末端节点。
4. 节点度：一个节点拥有的子节点数量。
5. 层：根为第 1 层，依次向下递增。

## 定义二叉树
-它可以没有根节点，作为一个空树存在
-如果不是空树，那么必须由根节点开始，左右子树组成，且左右子树都是二叉树


## 树的关键概念
-树的层次计算规则
 根节点为第1层，其子节点为第2层，依次向下递增
-树的高度
 叶子节点高度计为1，每向上一层，高度加1，直到根节点
-度的概念
 一个节点开叉出去多少子树，就是节点的度
-叶子节点
 度为0的节点，就是叶子节点，最后一层都是叶子节点

二叉树不可以被简单定义为每个节点度≤2 的普通树
普通树：子节点之间无顺序区分
左右子树的位置都是严格约束的，不能交换的
